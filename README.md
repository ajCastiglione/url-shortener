# URL Shortener service

This is a simple service to create short urls. The sole purpose is meant to shrink URLs for free, not meant to reinvent the wheel.

## Dependencies

<ul>
<li>Express</li>
<li>Mongoose / MongoDB</li>
<li>Config</li>
<li>valid-url</li>
<li>short-id</li>
</ul>
